# Name: Show top network sources by traffic volume (process)

Description: Capture and analyse network traffic received for the specified process in a specified time period, showing the IP addresses and DNS names of the remote systems, the port used, the amount of data received in the capture period.
Arguments:
  Capture Time - the number of seconds to run a network capture for (default 15 seconds)
  Show top - show only the top n sources (default is 10)
