# Name: AZ Change machine disk type

Description: Change the type of disk assigned to the Azure VM. Valid disk types are:

  Standard_LRS
  Premium_LRS
  StandardSSD_LRS
  UltraSSD_LRS
  Premium_ZRS
  StandardSSD_ZRS

Note that costs of this VM once updated could be very different.
